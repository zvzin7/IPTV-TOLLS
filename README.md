🚀 Como usar

Termux (Android)
```bash
pkg update && pkg upgrade
pkg install python git
git clone https://github.com/zvzin7/IPTV-TOLLS.git
cd IPTV-TOLLS
python bot.py
```

Linux

```bash
sudo apt update && sudo apt install python3 git
git clone https://github.com/zvzin7/IPTV-TOLLS.git
cd IPTV-TOLLS
python3 bot.py
```

Windows

```bash
git clone https://github.com/zvzin7/IPTV-TOLLS.git
cd IPTV-TOLLS
python bot.py
```

As dependências instalam automaticamente na primeira execução.
